# HXH
我在大学时写的作业。（打砖块）
运行环境：
pygame 2.6.1 (SDL 2.28.4, Python 3.10.11)
打开main.py文件后在控制台输入python main.py运行

规则：
道具颜色
名称（类型）
效果描述
▮▮▮ 绿色
​​扩展 (Expand)​​
挡板长度增加50%，持续10秒
▮▮▮ 橙色
​​加速 (Speed)​​
球速提升50%，持续8秒
▮▮▮ 粉色
​​生命 (Life)​​
生命值+1（上限3点）
▮▮▮ 红色
缩短 (shrink)
挡板长度减少50%，持续10秒
▮▮▮ 蓝色
​​减速 (Slow)​​
球速降低50%，持续8秒
